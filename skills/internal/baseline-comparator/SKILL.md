# Baseline Comparator

## Workflow

1. Compare the current run to:
   - the internal baseline
   - prior attempts for the same idea
   - any explicitly cited public references
2. Separate quantitative deltas from qualitative observations.
3. Highlight where the idea is novel even if raw metrics are mixed.

## Required output

- `metric deltas`
- `trend versus prior attempts`
- `novelty-preserving interpretation`

## Guardrails

- Comparisons must inform judgment, not force imitation.
- Say when the current data is too weak for strong claims.
